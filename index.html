🔗 เชื่อม Google Sheets
วิธีสร้าง Apps Script Proxy:
1. เปิด Google Sheet → Extensions → Apps Script
2. ลบโค้ดเดิม แล้ววางโค้ดด้านล่าง
3. กด Deploy → New deployment → Web app
4. Execute as: Me · Access: Anyone
5. Copy URL ที่ได้ไปใส่ในขั้นตอน ②
วาง Apps Script Web App URL และ Sheet ID
Sheet ID อยู่ใน URL ของ Sheet: /spreadsheets/d/ID/edit
